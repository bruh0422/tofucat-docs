# 私人頻道系統

用法 `=ticket`\
\
創建的私人頻道權限會繼承入口所屬類別的權限，這代表可以讓指定的成員或身分組查看創建的頻道。

{% tabs %}
{% tab title="創建私人頻道系統入口" %}
用法 `=ticket set`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理頻道</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}

{% tab title="刪除私人頻道系統" %}
用法 `=ticket delete/del`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理頻道</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}
{% endtabs %}
