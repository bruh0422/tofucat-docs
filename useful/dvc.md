# 動態語音頻道

用法 `=dvc`\
\
進入動態語音頻道後會創建一個新語音頻道並將成員移動至創建的頻道，當頻道沒人後會刪除該頻道。

{% tabs %}
{% tab title="創建動態語音頻道入口" %}
用法 `=dvc set`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理頻道</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}

{% tab title="刪除動態語音頻道" %}
用法 `=dvc delete/del`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理頻道</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}
{% endtabs %}
