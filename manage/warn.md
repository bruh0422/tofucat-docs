# 警告成員

用法 `=warn <成員>`\
\
警告成員並不會發生任何事，只是方便管理員註記。管理員們可以自行訂定規則，例如警告達到 3 個之後[禁言](timeout.md)該成員。

{% hint style="info" %}
你必須有 <mark style="color:green;">管理者</mark> 的權限才可執行此指令。
{% endhint %}

{% tabs %}
{% tab title="清除指定成員的警告" %}
用法 `=warn clear <成員>`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理者</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}

{% tab title="顯示指定成員的警告數" %}
用法 `=warn show <成員>`
{% endtab %}

{% tab title="顯示所有警告" %}
用法 `=warn list`
{% endtab %}
{% endtabs %}
