# 單一反應身分組

用法 `=unique_role/ur`\
\
此指令與[反應身分組](reaction\_role.md)相似，但是成員一次只能選擇一個身分組來獲取。

{% tabs %}
{% tab title="新增單一反應身分組" %}
用法 `=unique_role/ur add <頻道> <身分組>`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理身分組</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}

{% tab title="移除單一反應身分組" %}
用法 `=unique_role/ur remove <訊息ID>`

{% hint style="info" %}
你必須有 <mark style="color:green;">管理身分組</mark> 的權限才可執行此指令。
{% endhint %}
{% endtab %}
{% endtabs %}
