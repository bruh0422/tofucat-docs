# 工作賺錢

用法 `=work`\
\
一次工作可獲得的錢是依照你的[銀行](bank.md)總等級來計算，如下表：

| 銀行總等級 | 獲得           |
| ----- | ------------ |
| 0     | 50 \~ 500 元  |
| 10    | 100 \~ 500 元 |
| 20    | 150 \~ 500 元 |
| 30    | 200 \~ 500 元 |
| 40    | 250 \~ 500 元 |
| 50    | 300 \~ 500 元 |

{% hint style="info" %}
如果顯示處罰將在XX結束，那代表你的[銀行](bank.md)爆了。
{% endhint %}
