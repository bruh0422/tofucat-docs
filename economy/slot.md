# 玩拉霸機

用法 `=slot <金額>`\
\
系統會依下表來計算贏得的金錢：

| 圖案種數 | 獲得金錢                       |
| ---- | -------------------------- |
| 5、4  | -投入金額\*1 (投入金額的 1/10 進入獎池) |
| 3    | 投入金額\*0.5                  |
| 2    | 投入金額\*3                    |
| 1    | 獎池內獎金或投入金額\*10             |

想要得到獎池內的錢，必須同時滿足以下條件，否則將會得到投入金額\*10：

1. 投入金額 >= 300
2. 投入金額\*10 < 獎池的錢

{% hint style="info" %}
如果顯示處罰將在XX結束，那代表你的[銀行](bank.md)爆了。
{% endhint %}
